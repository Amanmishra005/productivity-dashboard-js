# Productivity Dashboard

This is a simple Productivity Dashboard project built using HTML, CSS, and JavaScript.

## Features
- Live Clock and Date
- To-Do List Management
- Mark Tasks as Completed
- Task Deletion
- LocalStorage Support (Tasks remain after page reload)

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)

## Purpose
This project was created as a learning mini-project to practice DOM manipulation and LocalStorage concepts.

## Future Improvements
- Dark Mode
- Task Editing Feature
- Drag & Drop Task Sorting
